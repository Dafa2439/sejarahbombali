## Deskripsi

Website ini merupakan proyek dokumentasi digital yang menyajikan informasi komprehensif mengenai Tragedi Bom Bali I tahun 2002. Dibangun sebagai media edukasi sejarah dengan pendekatan desain editorial modern, website ini mencakup berbagai aspek peristiwa mulai dari pengertian, latar belakang, dampak, hingga tokoh-tokoh kunci yang terlibat.

## Fitur Utama

| Fitur | Deskripsi |
|:------|:----------|
| **Navigasi Tab** | Sistem tab interaktif untuk berpindah antar-seksi konten |
| **Hero Slider** | Carousel otomatis menampilkan foto-foto dokumentasi peristiwa |
| **Timeline** | Visualisasi kronologis faktor-faktor latar belakang |
| **Impact Cards** | Kartu statistik dampak peristiwa dengan data kuantitatif |
| **Profil Tokoh** | Daftar tokoh kunci disertai foto dan deskripsi peran |
| **Berita & Sumber** | Kumpulan referensi dari media terpercaya dengan tautan langsung |
| **Responsif** | Tampilan optimal di desktop maupun perangkat mobile |

## Cara Menjalankan

1. Clone repository ini
   ```bash
   git clone https://github.com/Dafa2439/sejarahbombali.git
   ```
2. Buka file `index.html` di browser
   ```bash
   cd sejarahbombali
   open index.html
   ```
   Atau gunakan ekstensi **Live Server** di VS Code untuk pengalaman pengembangan yang lebih baik.

## Konten Website

| Seksi | Deskripsi |
|:------|:----------|
| **01 / Pengertian** | Penjelasan mengenai peristiwa Bom Bali I dan data korban |
| **02 / Latar Belakang** | Faktor-faktor penyebab: konflik Poso & Ambon, intervensi AS, pemilihan Bali, ideologi pelaku, teknis pelaksanaan |
| **03 / Dampak** | Dampak kemanusiaan, pariwisata & ekonomi, sosial, hukum, dan keamanan nasional |
| **04 / Tokoh** | Profil 5 tokoh kunci: Amrozi, Imam Samudra, Ali Gufron, Ali Imron, Hambali |
| **05 / Berita & Sumber** | Referensi dari Kompas, BBC, Tempo, Reuters, CNN Indonesia |
